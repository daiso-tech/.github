# @daiso-tech/core GitHub Organization

Welcome to the official GitHub organization for @daiso-tech/core.

We maintain a collection of open-source TypeScript libraries and tools for building modern, scalable, and type-safe backend systems. Our projects are designed to be modular, framework-agnostic, and easy to integrate into a variety of environments.

---

## About

@daiso-tech/core is focused on developer productivity, reliability, and modern best practices. We provide solutions for common backend challenges, with a strong emphasis on type safety and ease of use.

- Explore our [documentation](https://daiso-tech.dev/docs/installation) for guides and API references.
- Browse our [GitHub organization](https://github.com/daiso-tech) to discover all available packages.
